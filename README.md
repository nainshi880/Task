# UI Design (Static)

This folder contains a static HTML/CSS implementation of the provided landing page design.

## Run

Option A (quick): open `index.html` directly in your browser.

Option B (recommended): serve locally:

```bash
cd UI_design
npx serve .
```

## Pixel Perfect Workflow

1. Export your Figma frame as `reference.png`.
2. Put it in this folder (`UI_design/reference.png`).
3. Open the page and use the bottom-right overlay panel:
   - Toggle overlay on/off
   - Adjust opacity
   - Shortcut: press `O`
